# Pundo

## 植物空間點位標記工具  
## Plant Spatial Points Tool

**Pundo** 是一款專為森林生態學、森林動態樣區調查與植株空間資料建檔所設計的輕量級網頁工具。研究人員可以在樣區底圖上快速標記樹木或木本植物的空間座標，並同步整理物種、胸高直徑（DBH）與樣方資訊等調查資料。

本工具採用純前端設計，不需要架設資料庫或伺服器。使用者只要下載 HTML 檔案，即可在本機離線使用；也可以透過 GitHub Pages 線上開啟。所有資料皆保留在使用者自己的電腦中，適合用於樣區圖數位化、每木調查資料整理，以及森林樣區空間資料管理。

---

## 工具概念

Pundo 以森林樣區或子樣方的西南角作為相對座標原點 **(0, 0)**，並依照每棵植物在樣區中的相對位置進行座標標定。

若使用者已知樣區西南角的絕對座標，便可進一步將每棵植物的相對座標轉換為大地座標。此方法可減少逐棵植物使用 GPS 定位時可能產生的誤差，特別適合森林樣區、固定樣區或長期生態監測樣區使用。

---

## 主要功能

### 純前端運行

Pundo 不需要資料庫、伺服器或額外安裝軟體。使用者只需透過支援 HTML5 的瀏覽器開啟工具，即可進行資料匯入、點位標記、底圖校正與資料輸出。

資料會保留在使用者自己的電腦中，不會上傳至遠端伺服器，因此可提升調查資料的安全性與隱私性。

### 支援底圖空間校正

使用者可以匯入 **JPG、PNG 或 PDF** 格式的樣區掃描圖、手繪圖或電子化底圖。

工具提供底圖邊界微調、旋轉角度設定、黑框自動偵測與 buffer 保留功能，可協助使用者將圖紙上的樣區外框對齊數位畫布與座標網格，提升後續點位標記的準確度。

### 互動式點位標記

使用者可直接在畫布上以滑鼠點擊方式標記植物位置。系統會記錄該植物的空間座標，並可依照 DBH 數值自動調整點位圖示大小，使植株空間分布與大小結構更直觀呈現。

### CSV 資料匯出

完成點位標記後，使用者可以直接將資料下載為標準 CSV 格式。匯出的資料可用於後續統計分析、空間分析、GIS 製圖或森林動態樣區資料管理。

### 野外調查用 PDF 匯出

完成點位標記後，使用者可以將樣區點位圖輸出為 PDF 檔案，作為後續野外複查、樣區調查或資料校對使用。

## 使用流程

## 1. 資料上傳

若使用者已經有植物調查資料，可透過「資料上傳」功能匯入 CSV 檔案。系統支援 **UTF-8** 與 **Big5** 編碼格式。

以森林動態樣區資料為例，匯入資料可包含下列欄位：

| 欄位名稱 | 說明 |
|---|---|
| `Tag` | 植株編號，必要欄位 |
| `plot` | 樣區名稱 |
| `x1` | 10 × 10 m 樣方的 X 座標 |
| `y1` | 10 × 10 m 樣方的 Y 座標 |
| `x2` | 5 × 5 m 子樣方的 X 座標 |
| `y2` | 5 × 5 m 子樣方的 Y 座標 |
| `Sp` | 物種名稱 |
| `DBH` | 胸高直徑，單位依原始資料設定 |
| `x3` | 植株在子樣方中的精細 X 座標 |
| `y3` | 植株在子樣方中的精細 Y 座標 |

其中，**`Tag` 為必要欄位**。若匯入資料中沒有 `Tag` 欄位，系統將無法進行後續點位標記與資料更新。

其他欄位則可視資料狀況彈性使用。例如，若使用者處理的是零散樣區或非森林動態樣區資料，資料中沒有 `x1`、`y1`、`x2`、`y2` 等欄位，仍可進入系統並進行點位標記。

---

## 2. 植株點位資料設定

在「植株點位」區塊中，使用者可輸入或選擇目標植株的 `Tag`。

若該 `Tag` 已存在於匯入資料中，系統會自動帶入既有欄位資訊，例如樣區名稱、物種名稱、DBH 與樣方位置。

若該 `Tag` 為新植株，使用者可手動填寫 `plot`、`Sp`、`DBH` 等資訊。

使用者也可以透過 `plot`、`x1`、`y1`、`x2`、`y2` 等欄位篩選特定樣區或樣方，以便逐區完成植株點位標記。

---

## 3. 畫布設定與底圖校正

進入系統後，使用者需先設定樣區或子樣方的邊界大小，單位為 **cm**。樣區形狀不一定要為正方形，系統會依據使用者設定的 X 軸與 Y 軸範圍決定畫布比例與座標計算結果。

使用者也可以上傳樣區手繪圖、掃描圖、電子化底圖或 PDF 圖檔，並透過下列功能進行校正：

- 微調底圖邊界
- 調整底圖旋轉角度
- 自動偵測樣區外框
- 保留外框周圍 buffer
- 對齊畫布上的數位網格
- 對齊樣區外圍黑框或實際調查邊界
- 移除或置換底圖

完成校正後，底圖即可作為後續點位標記的參考。

---

## 4. 新增與更新點位

確認目標植株資料無誤後，點擊「準備點擊圖面」按鈕，接著在畫布上的對應位置點擊滑鼠，即可完成該植株的空間座標標記。

若該 `Tag` 已存在，系統會更新該筆資料的 `x3` 與 `y3` 座標。

若該 `Tag` 不存在，系統會新增一筆資料，並記錄使用者輸入的 `plot`、`Sp`、`DBH` 與座標資訊。

系統會將點擊位置轉換為對應的空間座標，並將精細座標紀錄於 `x3`、`y3` 兩個欄位中。這兩個欄位表示植株相對於樣方左下角或西南角的距離，單位為 **cm**。

---

## 5. 資料匯出

完成所有植物點位標記與資料確認後，使用者可點擊 CSV 匯出按鈕下載資料。

匯出的 CSV 檔案將包含原始調查資料，以及新增或更新後的完整空間座標欄位，例如：

| 欄位名稱 | 說明 |
|---|---|
| `x3` | 植株在樣方中的精細 X 座標 |
| `y3` | 植株在樣方中的精細 Y 座標 |

匯出後的資料可直接用於 R、Python、QGIS、ArcGIS 或其他統計與空間分析軟體。

---

## 6. PDF 檔案匯出

完成植物位置標記後，使用者可以輸出野外調查用 PDF 點位圖。

系統可依照目前篩選條件輸出特定樣區或樣方，也可以批次輸出多個樣方的點位圖。PDF 圖面可用於：

- 野外複查
- 樣區補測
- 植株定位
- 調查資料校對
- 紙本紀錄與數位資料比對

## 系統需求

Pundo 可於任何支援 HTML5 的現代網頁瀏覽器中執行，例如：

- Google Chrome
- Microsoft Edge
- Safari
- Firefox

本機版 HTML 檔案不需額外安裝軟體，也不需網路連線即可使用。

若透過 GitHub Pages 或 CDN 載入外部 JavaScript 套件，則需要網路連線。

## 適用情境

Pundo 適合用於下列工作：

- 森林樣區手繪圖數位化
- 木本植物空間點位標記
- 森林動態樣區每木資料建檔
- 樣區座標資料整理
- 相對座標轉換為大地座標前的資料準備
- 長期生態監測樣區資料管理
- 野外調查資料後處理
- 樣區複查與點位校對

---

## 簡要說明

Pundo 是一個以森林樣區相對座標為核心的植物空間點位標記工具。它讓研究人員能在掃描圖、手繪圖、PDF 或電子化底圖上快速標定植株位置，並整合 `Tag`、物種、DBH 與座標資料。

透過純前端、可離線使用、CSV 匯出與 PDF 點位圖輸出的設計，Pundo 可協助森林生態研究者將紙本或影像化樣區圖轉換為可分析、可保存且可重複使用的數位空間資料。
