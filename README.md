# Pundo - 植物空間點位標記工具 (Plant Spatial Points Tool)

這是一個專為森林生態學與樣區調查設計的輕量級網頁工具。協助研究人員在樣區底圖上快速標記樹木的空間座標、物種與胸高直徑 (DBH) 等重要資訊。

本工具完全獨立於瀏覽器前端運行，無需架設資料庫或伺服器，支援離線使用與跨平台分享，非常適合將手繪樣區圖數位化，或是進行長期的森林動態樣區資料建檔與管理。

## 主要功能

* **純前端運行**：下載 HTML 檔案即可離線使用，或透過 GitHub Pages 線上開啟，資料完全保留在本地端，確保調查數據安全。
* **底圖空間校正 (Georeference)**：支援匯入 JPG 或 PNG 樣區圖紙，並可自由調整實際物理邊界與旋轉角度，精準對齊數位網格。
* **互動式點位標記**：滑鼠點擊即可記錄空間座標，並根據 DBH 數值自動縮放點位圖示大小，直觀呈現林木的空間分佈結構。
* **智慧連動輸入**：支援匯入 CSV 檔案，輸入既有 Tag 時會自動帶入樣區 (Plot)、物種 (Sp)、DBH 以及原有座標 (x1, y1, x2, y2) 等資訊，大幅減少重複輸入的時間。
* **一鍵匯出**：標記完成後，可直接下載為標準 CSV 格式，方便後續匯入 R 軟體進行枯死木動態、樹高估算或生質量的建模與空間分析。

## 使用流程

### 1. 資料上傳與畫布設定

首先設定樣區的實際 X 軸與 Y 軸最大範圍 (單位為 cm)。如果已經有部分建檔的資料，可以透過「資料上傳」匯入 UTF-8 或 Big5 格式的 CSV 檔案。

### 2. 底圖校正

上傳樣區的手繪稿或掃描圖，並透過下方的「微調底圖邊界」與「旋轉角度」功能，將圖紙上的外圍黑框精準對齊畫布上的數位網格。

### 3. 新增與更新點位

在「新增 / 更新點位」區塊輸入目標樹木的 Tag。若是既有資料，系統會自動帶入其他欄位；若是新樹木，請手動填寫 Plot、Species、DBH 等資訊。

確認資訊無誤後，點擊「準備點擊圖面」按鈕，接著在畫布上的對應位置點擊滑鼠，即可完成空間點位的座標標記。

### 4. 資料匯出

完成所有樹木的標記與確認後，點擊右下角的「下載完成的 CSV 檔案」按鈕，即可取得包含完整空間座標 (x3, y3) 的資料表。

## 系統需求

只需要任何支援 HTML5 的現代網頁瀏覽器 (如 Google Chrome, Microsoft Edge, Safari, Firefox) 即可順暢執行。
